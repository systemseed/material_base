Grid
====

Grid system uses [Flexbox Grid](http://flexboxgrid.com/).

Flexbox Grid implements Bootstrap-like approach, uses same markup and class names.

Exemples:

~~~
<div class="row">
  <div class="col-xs-12 col-sm-8 col-md-6 col-lg-4">
    <div>Content 1</div>
  </div>
  <div class="col-xs-12 col-sm-8 col-md-6 col-lg-4">
    <div>Content 2</div>
  </div>
  <div class="col-xs-12 col-sm-8 col-md-6 col-lg-4">
    <div>Content 3</div>
  </div>
</div>
~~~
